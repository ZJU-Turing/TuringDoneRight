# 数学分析（甲）Ⅱ（H）

## 课程概要
本课程为大一下预置课，使用陈纪修版《数学分析（下）》作为教材，主要内容包括数项级数、函数项级数、傅里叶级数，多元函数的极限、连续性与微分，重积分、曲线积分、曲面积分等。

相比数分一，数分二不那么看重证明，重心更多地放在计算上，期末考试中计算题足足占百分之六七十。
