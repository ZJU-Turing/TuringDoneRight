# 线性代数 Ⅱ（H）

## 课程概要
本课程为大一下预置课，使用《线性代数应该这样学》（《Linear Algebra Done Right》）作为教材，主要内容包括对偶、线性泛函、内积空间、自伴算子与正规算子、谱定理、广义本征向量、极小多项式、若尔当形、算子的分解、算子的复化等。这本教材前半部分与上学期学习内容重合较多，下半部分重点在于算子理论。总体而言，本课程相较上学期更加“抽象”，也更加深入线性代数的本质，难度较高。
