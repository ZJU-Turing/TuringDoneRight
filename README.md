> [!Warning]
> 正在建设中

## 本地构建

- 安装依赖
    ```sh
    $ pip install -r requirements.txt
    ```
- 开启本地预览服务
    ```sh
    $ mkdocs serve # Serving on http://127.0.0.1:8000/
    ```

## 修改发布
